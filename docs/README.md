# Tellurian ForgeWorks IDE Documentation

Welcome to the official documentation for the **Tellurian ForgeWorks IDE**.

The ForgeWorks IDE is a custom, branded distribution of VSCodium, designed to provide a secure, policy-compliant, and pre-configured development environment for all Tellurian Corp engineers.
